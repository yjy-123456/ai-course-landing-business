# AI 课程报名页业务工具包

这个项目是一套用于启动“AI 课程报名页/预约页搭建服务”的实用资料包，面向 AI 课程老师、培训机构、咨询顾问和知识型创作者。

## 这个业务卖什么

服务内容：

> 为 AI 课程、训练营、培训项目和咨询服务搭建课程报名页。

客户得到的结果：

- 课程卖点表达更清楚
- 手机端浏览体验更好
- 更容易收集报名线索
- 有明确的微信咨询入口
- 不做复杂后台也能快速上线

## 推荐的第一个套餐

基础课程报名页：999 元

包含：

- 1 个响应式报名页
- 课程介绍
- 适合人群
- 课程安排
- 常见问题
- 报名表单或嵌入式表单
- 微信二维码入口
- 部署上线
- 1 次小范围修改

交付周期：

- 2-3 天

## 第一版不做后台

第一版不建议做自定义后台和数据库。

建议使用：

- React + Vite 制作页面
- Cloudflare Pages 或 Vercel 部署
- 飞书表单、金数据或腾讯问卷收集报名数据
- 微信二维码承接咨询

这样可以控制成本、缩短交付周期，也更容易让客户理解项目边界。

## 运行 Demo

进入 `demo` 目录：

```bash
npm install
npm run dev
```

然后打开 Vite 输出的本地地址。

构建部署版本：

```bash
npm run build
```

## 今日行动清单

1. 阅读 `docs/01-complete-plan.md`。
2. 打开 `docs/02-client-materials.md`，发布第一条熟人圈动态。
3. 给 20 个可能认识课程老师或咨询顾问的人发送私信话术。
4. 如有需要，把 demo 内容替换成你更想切入的细分领域。
5. 部署 demo 页面，把它作为第一个作品案例。

- 发送 20 条私信

## 客户边界

第一版包含：

- 报名页
- 表单集成
- 微信咨询入口
- 静态部署

第一版不包含：

- 登录系统
- 支付系统
- 管理后台
- 学员管理
- CRM
- 短信通知
- 复杂后端

如果客户需要这些功能，可以按定制项目报价，起步价建议 3999 元以上。
